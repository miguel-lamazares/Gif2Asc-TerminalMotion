## インストール方法

#### Docker 🐳
依存関係を手動で管理したくない場合に推奨されます。

```bash
git clone https://github.com/miguel-lamazares/Gif2Asc-TerminalMotion.git
cd Gif2Asc-TerminalMotion/Gif2Asc/Docker
docker build -t gif2asc .
docker run -it gif2asc
```

これにより完全な実行環境が作成され、
Python、Java、バージョン管理などの問題を回避できます。

#### Git 🧬
すべてをローカル環境で実行したい方向けです。

1. リポジトリをクローン
```bash
https://github.com/miguel-lamazares/Gif2Asc-TerminalMotion.git
cd Gif2Asc-TerminalMotion
```

2. Python の依存関係をインストール
```bash
pip install pillow requests
pip install Gif2Asc-TerminalMotion/Gif2Asc/Engine/MidiaConvertion/TerminalLib -e .
```

3. システム依存関係をインストール
- MPV（ターミナルでアニメーションを再生するために必要）
```bash
sudo apt install mpv
```

- ローカル実行に必要なすべての依存関係をインストールする場合
```bash
sudo apt install mpv python java-11-openjdk gcc jp2a git
```

- Docker を使用して実行する場合
```bash
sudo apt install mpv python java-11-openjdk gcc jp2a git docker
```

または、リポジトリをクローン後、
start.sh を実行してすべての依存関係を自動インストールできます。

4. プロジェクトを実行

### トラブルシューティング

* pip の権限エラーが発生する場合：
  `pip install --user ...` を使用するか、
  仮想環境を作成してください。
  (`python -m venv venv && source venv/bin/activate`)

* `mpv` コマンドが見つからない場合：
  パッケージが正しくインストールされ、
  PATH に含まれているか確認してください。

* Java の依存関係に問題がある場合：
  `java -version` でインストールされているバージョンを確認してください。

#### シェルコマンド一覧

- Start:
  Gif2Asc/Starters/Linux/install dependences/start.sh
  すべての依存関係を自動的にインストールします。
  Kali Linux や制限のあるディストリビューションでは、
  venv の使用を推奨します。

- Full Process:
  Gif2Asc/Starters/Linux/full_process.sh
  ダウンロード、変換、レンダリング、実行までの
  完全な処理を行います。

- Quick Start:
   Gif2Asc/Starters/Linux/quick_start.sh
   カスタマイズなしで、
   前回の設定を使用して実行します。

- Execute Last:
  Gif2Asc/Starters/Linux/execute_last.sh
  メモリ上の最後のファイルを実行し、
  アニメーションと音声を再生します。

## 依存関係一覧

### Python 🐍

| 依存関係 | 用途 | インストール方法 |
|---------|------|------------------|
| Pillow | 画像処理（読み込み、編集、保存） | pip install pillow |
| Requests | HTTP リクエスト（ファイルダウンロードなど） | pip install requests |
| TerminalLib | ターミナル描画ライブラリ（プロジェクト内） | pip install Gif2Asc-TerminalMotion/Gif2Asc/Engine/MidiaConvertion/TerminalLib -e . |

### システム 👨🏼‍💻

| 依存関係 | 用途 | インストール方法 |
|---------|------|------------------|
| MPV | ターミナルで音声を再生 | sudo apt install mpv |
| Python 3.x | Pythonコードの実行 | sudo apt install python |
| Java JDK 11+ | Java製エンジンのコンパイル・実行 | sudo apt install openjdk-11-jdk |
| GCC | エンジンコンポーネントのコンパイル | sudo apt install gcc |
| JP2A | アニメーションフレームの変換 | sudo apt install jp2a |
| Git | リポジトリ管理 | sudo apt install git |
| Docker（任意） | 環境を完全に分離 | sudo apt install docker.io |
